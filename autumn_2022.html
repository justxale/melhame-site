Уважаемые студенты!

Для проверки рейтинга и контроля оценок по итогам сессии необходимо использовать сервис "Зачетная книжка". По окончании сессии деканаты закроют ведомости, и в ЛК студента будут показаны итоговые оценки по промежуточной аттестации. Используйте сервис "Сервис информирования студента о баллах БРС" для контроля текущей аттестации.
Внимание! Итоговая оценка считается с учетом всех контрольных мероприятий в семестре. Положительная оценка по дисциплине будет отображаться только после успешной сдачи всех контрольных мероприятий.
